dlink-backup
============

backup dlink switches


    $ ./dlink.exp host user pass config_name tftp_host


Supported hardware
------------------


### D-Link


* dlink DES-3052
* dlink DES-3200-10
* dlink DES-3200-26
* dlink DES-3200-28
* dlink DES-3526
* dlink DES-3552
* dlink DGS-3312SR

### SNR

* SNR-S2950-24G
